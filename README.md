# Weather App

Простая программа на Python для получения текущей погоды через OpenWeather API.

## Возможности :

- поиск города по названию и коду страны
- температура
- ощущаемая температура
- описание погоды
- влажность
- скорость ветра

## Запуск программы

1. Установите нужную библиотеку:

pip install requests

2. Запустите программу:


python main.py

3.Введите название города и код страны, после чего программа выведет информацию о погоде.
